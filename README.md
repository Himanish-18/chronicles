# Multi-Service Blog Deployment

## Overview
A production-ready blogging platform showcasing a modern DevOps workflow. It consists of a React frontend, a Node.js/Express backend, and a MySQL database, all orchestrated via Docker Compose and deployed on AWS EC2 using GitHub Actions for CI/CD.
